Instructions for installing each module required

Currently using Python 3.8 and Node V12.18.2

Windows:
    NodeJS:
        Go to https://nodejs.org/en/download/ and click on Windows Installer
    ExpressJS:
        Go to current directory and run 'node init'
        Run 'npm i express --save'
    File System:
        Run 'npm i fs' in current directory

    PyAudio:
        Go to https://www.lfd.uci.edu/~gohlke/pythonlibs/#pyaudio
        Download the 'PyAudio‑0.2.11‑cp38‑cp38‑win32.whl' file
        Go to command prompt, change directory to where the file is located, and do 'pip install PyAudio‑0.2.11‑cp38‑cp38‑win32.whl'
    gTTS:
        'pip install gtts'
    Playsound:
        'pip install playsound'
    Speech-recognition:
        'pip install SpeechRecognition'
    pyautogui:
        'pip install PyAutoGUI'
